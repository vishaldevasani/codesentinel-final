Setup Guide
1
Install dependencies: pip install -r requirements.txt
2
Copy .env.example → .env and fill ANTHROPIC_API_KEY + GITHUB_TOKEN
3
Start server: uvicorn main:app --reload
4
Expose with ngrok: ngrok http 8000
5
Add the webhook URL above in GitHub repo → Settings → Webhooks
6
Open a PR — CodeSentinel reviews it automatically!
